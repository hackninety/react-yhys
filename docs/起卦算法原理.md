# 皇极经世起卦算法原理

本项目依据北宋哲学家邵雍的《皇极经世》及后世黄畿等学者的正统推衍法则，构建了完整的元、会、运、世、年五级时间单位的卦象推衍系统。核心思想在于**“元统会，会统运，运统世，世统年”**的层层嵌套、相生统辖逻辑。

---

## 1. 辟卦与运卦（会统运）

- **辟卦（十二消息卦）**：一元十二会，每会对应一个辟卦（例如子会为复卦、丑会为临卦等）。
- **运卦的衍生**：一会管辖30个运。这30个运被平均分为5组（每组6个运），按顺序对应先天圆图剔除四正卦后的主卦。
  - 具体到某一运的**“值运卦”**，是由其对应的主卦，根据该运在组内的顺位（第1至第6），依次由主卦变动对应爻（初爻至上爻）所得。

## 2. 世卦（运统世）

一运统管12世，共计360年。在正统算法中，世卦是由运卦统领生成，而非固定不变的。
- 考虑到每2世合为60年（一甲子），我们将这2世作为一个单元，称为一个**“甲子世”**。因此一运共有6个甲子世。
- **推演规则**：某一世的“世卦”，是由其上层统辖的“运卦”，根据它属于第几个甲子世（1至6），变动运卦对应的爻位（初爻至上爻）而派生出来。
  - 第1、2世（第1个甲子）：运卦变初爻得世卦。
  - 第3、4世（第2个甲子）：运卦变二爻得世卦。
  - 依此类推……

## 3. 年/岁卦（世统年）

年卦（值年卦）代表了具体的流年气象，它由当下所处的“世卦”进一步变动派生，实现“世统年”。
- **推演规则**：以1984年（近期的一个甲子年）为六十甲子的锚点，计算当前年份在60年循环中的偏移（0-59）。
- 依据该偏移量，按照 `(偏移量 % 6) + 1` 的规则确定变爻位（初爻到上爻，6年一循环）。
- **四正卦避让（退格机制）**：乾、坤、坎、离在皇极经世体系中作为闰卦不用。如果在变爻过程中恰好得到四正卦之一，算法会自动顺推尝试下一爻，直至变出一个非四正卦作为当年的有效值年卦。

## 4. 月卦与日卦

月、日的推衍目前采用标准的六十甲子流转配合先天六十卦序列（剔除四正卦）进行：
- **月卦**：“日甲月子，合乎为复”。以甲子年子月（1984年12月）为复卦起始，每月向前推进一卦。
- **日卦**：以甲子日对应复卦为基准，按照先天六十卦序列逐日流转循环。
