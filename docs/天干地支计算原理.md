# 天干地支计算原理

本文档详细说明皇极经世历与夏历（农历/八字）在天干地支计算上的异同。

## 概述

| 历法 | 换年节点 | 月份起始 | 日干支 | 时干支 |
|------|---------|---------|--------|--------|
| **皇极经世历** | 冬至 | 冬至起子月 | 六十甲子循环 | 日上起时法 |
| **夏历（八字）** | 立春 | 大雪起子月 | 六十甲子循环 | 日上起时法 |

---

## 一、年干支

### 1.1 皇极经世历

**换年节点：冬至**

皇极经世历以冬至为新年的起点。冬至之后即进入下一年。

```
示例：
- 2025年12月21日（冬至）之前 → 乙巳年（皇极第69042年）
- 2025年12月21日（冬至）及之后 → 丙午年（皇极第69043年）
```

**计算方法：**
```typescript
// 判断是否在冬至之后
const dongzhi = getTermStartDate(gregorianYear, 23) // 冬至节气索引=23
const isAfterDongzhi = currentDate >= dongzhi

// 皇极年对应的公历年
const huangjiGregorianYear = isAfterDongzhi ? gregorianYear + 1 : gregorianYear

// 年干支（1984年为甲子年）
const yearIndex = (huangjiGregorianYear - 1984) % 60
const yearGanZhi = 天干[yearIndex % 10] + 地支[yearIndex % 12]
```

### 1.2 夏历（八字）

**换年节点：立春**

夏历（八字命理）以立春为新年的起点。立春之后即进入下一年。

```
示例：
- 2025年2月3日（立春）之前 → 甲辰年
- 2025年2月3日（立春）及之后 → 乙巳年
```

**计算方法：**
```typescript
// 获取该年立春日期（节气索引2=立春）
const lichun = getTermStartDate(year, 2)

// 判断是否在立春之前
const ganzhiYear = currentDate < lichun ? year - 1 : year

// 年干支（1984年为甲子年）
const yearIndex = (ganzhiYear - 1984) % 60
const yearGanZhi = 天干[yearIndex % 10] + 地支[yearIndex % 12]
```

## 二、月干支

### 2.1 皇极经世历

**月份起始：冬至起子月**

皇极经世历以冬至为子月的开始，每两个节气为一个月。

| 皇极月 | 起始节气 | 终止节气 |
|--------|---------|---------|
| 子月（第1月） | 冬至 | 小寒结束 |
| 丑月（第2月） | 大寒 | 立春结束 |
| 寅月（第3月） | 雨水 | 惊蛰结束 |
| 卯月（第4月） | 春分 | 清明结束 |
| 辰月（第5月） | 谷雨 | 立夏结束 |
| 巳月（第6月） | 小满 | 芒种结束 |
| 午月（第7月） | 夏至 | 小暑结束 |
| 未月（第8月） | 大暑 | 立秋结束 |
| 申月（第9月） | 处暑 | 白露结束 |
| 酉月（第10月） | 秋分 | 寒露结束 |
| 戌月（第11月） | 霜降 | 立冬结束 |
| 亥月（第12月） | 小雪 | 大雪结束 |

**月干计算（年上起月法）：**

根据皇极年干确定寅月的天干，然后顺推：

| 年干 | 寅月天干 | 口诀 |
|------|---------|------|
| 甲、己 | 丙 | 甲己之年丙作首 |
| 乙、庚 | 戊 | 乙庚之岁戊为头 |
| 丙、辛 | 庚 | 丙辛必定寻庚起 |
| 丁、壬 | 壬 | 丁壬壬位顺行流 |
| 戊、癸 | 甲 | 戊癸之年甲寅始 |

```
示例（皇极丙午年）：
- 年干=丙 → 寅月起庚
- 子月相对寅月偏移10 → 月干=(庚+10)%10=庚
- 子月月干支=庚子
```

### 2.2 夏历（八字）

**月份起始：以"节"分月（大雪起子月）**

夏历以12个"节"（非"气"）作为月份的分界点：

| 夏历月 | 起始节气（节） | 月支 |
|--------|--------------|------|
| 子月 | 大雪 | 子 |
| 丑月 | 小寒 | 丑 |
| 寅月 | 立春 | 寅 |
| 卯月 | 惊蛰 | 卯 |
| 辰月 | 清明 | 辰 |
| 巳月 | 立夏 | 巳 |
| 午月 | 芒种 | 午 |
| 未月 | 小暑 | 未 |
| 申月 | 立秋 | 申 |
| 酉月 | 白露 | 酉 |
| 戌月 | 寒露 | 戌 |
| 亥月 | 立冬 | 亥 |

**月干计算：** 同皇极，使用年上起月法，但基于夏历年干。

```
示例（夏历乙巳年，大雪后）：
- 年干=乙 → 寅月起戊
- 子月相对寅月偏移10 → 月干=(戊+10)%10=戊
- 子月月干支=戊子
```

### 2.3 皇极与夏历月份对比

以2025年12月23日（冬至后第3天）为例：

| 历法 | 年干支 | 月干支 | 说明 |
|------|-------|--------|------|
| **皇极** | 丙午 | 庚子 | 冬至后=丙午年，冬至起子月，丙年寅月起庚 |
| **夏历** | 乙巳 | 戊子 | 立春前=乙巳年，大雪后=子月，乙年寅月起戊 |

## 三、日干支

### 日干支计算（两种历法一致）

日干支是连续的六十甲子循环，**与历法无关**，是天文周期。

**计算方法：**
```typescript
// 基准日：2000年1月1日 = 戊午日（索引54）
const BASE_DATE = new Date(2000, 0, 1)
const BASE_INDEX = 54

// 计算与基准日的天数差
const diffDays = Math.floor((targetDate - BASE_DATE) / (1000 * 60 * 60 * 24))

// 日干支索引
const dayIndex = (BASE_INDEX + diffDays) % 60
const dayGanZhi = 天干[dayIndex % 10] + 地支[dayIndex % 12]
```

**特点：**
- 日干支是连续循环的，不受年月划分影响
- 皇极经世历和夏历的日干支**完全一致**
- 这是因为日干支记录的是"太阳日"的周期

## 四、时干支

### 时辰划分（两种历法一致）

| 时辰 | 地支 | 时间范围 |
|------|-----|---------|
| 子时 | 子 | 23:00 - 01:00 |
| 丑时 | 丑 | 01:00 - 03:00 |
| 寅时 | 寅 | 03:00 - 05:00 |
| 卯时 | 卯 | 05:00 - 07:00 |
| 辰时 | 辰 | 07:00 - 09:00 |
| 巳时 | 巳 | 09:00 - 11:00 |
| 午时 | 午 | 11:00 - 13:00 |
| 未时 | 未 | 13:00 - 15:00 |
| 申时 | 申 | 15:00 - 17:00 |
| 酉时 | 酉 | 17:00 - 19:00 |
| 戌时 | 戌 | 19:00 - 21:00 |
| 亥时 | 亥 | 21:00 - 23:00 |

### 时干计算（日上起时法）

根据日干确定子时的天干，然后顺推：

| 日干 | 子时天干 | 口诀 |
|------|---------|------|
| 甲、己 | 甲 | 甲己还加甲 |
| 乙、庚 | 丙 | 乙庚丙作初 |
| 丙、辛 | 戊 | 丙辛从戊起 |
| 丁、壬 | 庚 | 丁壬庚子居 |
| 戊、癸 | 壬 | 戊癸何方发，壬子是真途 |

```
示例（丙寅日，卯时 05:00-07:00）：
- 日干=丙 → 子时起戊
- 卯时相对子时偏移3 → 时干=(戊+3)%10=辛
- 卯时时干支=辛卯
```

**注意：** 23:00后的子时属于次日，使用次日的日干计算。

## 五、月卦（十二消息卦）

### 5.1 十二消息卦概述

十二消息卦（又称十二辟卦）是《易经》中表示阴阳消长变化的十二个卦象，对应十二个月份。

"消"指阴气渐长、阳气渐消；"息"指阳气渐长、阴气渐息。

| 丑月 | 臨 | ䷒ | ⚊⚊⚋⚋⚋⚋ | 二阳 |
| 寅月 | 泰 | ䷊ | ⚊⚊⚊⚋⚋⚋ | 三阳（三阳开泰） |

| 辰月 | 夬 | ䷪ | ⚊⚊⚊⚊⚊⚋ | 五阳 |
| 巳月 | 乾 | ䷀ | ⚊⚊⚊⚊⚊⚊ | 六阳（纯阳） |
| 午月 | 姤 | ䷫ | ⚋⚊⚊⚊⚊⚊ | 一阴生（夏至） |

### 5.3 阴阳消长规律

```
冬至（子月）：一阳生于下 → 復卦
     ↓ 阳气渐长
夏至（午月）：纯阳至极 → 乾卦 → 一阴生于下 → 姤卦
     ↓ 阴气渐长
冬至（亥月末）：纯阴至极 → 坤卦 → 一阳复生 → 復卦
```

这体现了《易经》"物极必反、阴阳循环"的哲学思想。

## 六、运卦与世卦

皇极经世书中有两套不同层级的卦象推演系统：**运卦（星卦）**和**世卦（辰卦）**。

### 6.1 时间体系

| 层级 | 名称 | 数量/元 | 时长 |
|------|------|---------|------|
| 日（元） | 元 | 1 | 129600年 |
| 月（会） | 会 | 12 | 10800年/会 |
| 星（运） | 运 | 360 | 360年/运 |
| 辰（世） | 世 | 4320 | 30年/世 |
| 年（岁） | 岁 | 129600 | 1年/岁 |

### 6.2 四正卦的特殊处理

根据《皇极经世书解》原文，四正卦（乾、坤、坎、离）在元会运世体系中有特殊地位：

**核心原则**：
- **主序列剔除**：64卦共384爻，减去四正卦的24爻（4卦×6爻），得到360用数
- **四正卦不是弃用**：而是**另作"闰卦/闰爻"体系**，用来分主二十四气

**原文依据**（《皇极经世书解》卷十三）：
> "乾坤离坎的二十四策拿出去，剩下的就成360"
> "不用者，非不用也，用之以作闰卦"
> "以四闰卦分主二十四气，始以离，继以乾坎，终以坤"

**闰卦对应节气**：

| 闰卦 | 分主节气 | 说明 |
|------|---------|------|
| 离 | 春分、清明、谷雨、立夏、小满、芒种 | 始以离 |
| 乾 | 夏至、小暑、大暑、立秋、处暑、白露 | 继以乾 |
| 坎 | 秋分、寒露、霜降、立冬、小雪、大雪 | 继以坎 |
| 坤 | 冬至、小寒、大寒、立春、雨水、惊蛰 | 终以坤 |

### 6.3 运卦（星卦）

**适用层级**：会 → 运（每会30运）

**核心规则**：
1. 按伏羲先天六十四卦方圆图顺时针排序
2. 剔除四正卦（乾坤坎离）得到60卦
3. 60卦按先天序平均分配到12会，每会5卦
4. 每卦统领6运，5卦 × 6运/卦 = 30运/会

**原文依据**：
> "乾之数一，兑之数二，离之数三，震之数四，巽之数五，坎之数六，艮之数七，坤之数八，交相重而为六十四焉。乾、兑、离、震，在天为阳，在地为刚，在天则居东南，在地则居西北。巽、坎、艮、坤，在天为阴，在地为柔，在天则居西北，在地则居东南。"

**先天八卦数**：乾(1), 兑(2), 离(3), 震(4), 巽(5), 坎(6), 艮(7), 坤(8)

**"交相重"规则**：每个64卦用(上卦数,下卦数)表示

先天八卦数与二进制对应：
- 乾(1)=111, 兑(2)=011, 离(3)=101, 震(4)=001
- 巽(5)=110, 坎(6)=010, 艮(7)=100, 坤(8)=000

**先天64卦圆图顺序规则**：
```
【阳消阴长半圈】乾(1,1) → 姤(1,5) → ... → 剥(7,8) → 坤(8,8)
【阳长阴消半圈】坤(8,8) → 复(8,4) → ... → 夬(2,1) → 回到乾(1,1)
```

### 6.4 世卦（辰卦）

**适用层级**：运 → 世（每运12世）

**核心原则**："父生子，子生孙"
- 会（辟卦）→ 运（运卦）→ 世（世卦）
- 世卦由**运卦**爻变产生
- **剔除四正卦**（与运卦相同）

**计算规则**：
1. 首先确定该世所属的运，获取**运卦**
2. 对运卦进行爻变产生世卦
3. 遇到四正卦则跳过，继续爻变
4. 6个非四正卦 × 2世/卦 = 12世

**分配规则**：

| 世在运内序号 | 世卦来源 | 时长 |
|-------------|---------|------|
| 第1-2世 | 运卦爻变（非四正卦）第1个 | 60年 |
| 第3-4世 | 运卦爻变（非四正卦）第2个 | 60年 |
| 第5-6世 | 运卦爻变（非四正卦）第3个 | 60年 |
| 第7-8世 | 运卦爻变（非四正卦）第4个 | 60年 |
| 第9-10世 | 运卦爻变（非四正卦）第5个 | 60年 |
| 第11-12世 | 运卦爻变（非四正卦）第6个 | 60年 |

**大过运（午会第5运）世卦示例**：

大过卦（011110）爻变产生世卦（剔除四正卦）：

|--------|------|------|-----------|------|
| 第1-2世 | 初爻变 | 夬 | 否 | ✓ 夬 |
| 第3-4世 | 二爻变 | 咸 | 否 | ✓ 咸 |
| 第5-6世 | 三爻变 | 困 | 否 | ✓ 困 |
| 第7-8世 | 四爻变 | 井 | 否 | ✓ 井 |
| 第9-10世 | 五爻变 | 恒 | 否 | ✓ 恒 |
| 第11-12世 | 上爻变 | 姤 | 否 | ✓ 姤 |

### 6.5 岁卦（值年卦）

**适用层级**：每年一卦

**锚点规则**："日甲月子，合乎为复"
- 甲子年 = 复卦（子月消息卦，象征一阳来复）
- 复卦在先天60卦序中索引30（第31位）
- 正向循环：年份增加，卦象索引增加

**计算规则**：
1. 按先天六十四卦次序循环（剔除四正卦后60卦）
2. 每年对应一个卦，与六十甲子年一一对应
3. 锚点：甲子年(1984年) = 复卦（先天60卦序第31位，索引30）

**先天卦序（邵雍方圆图）**：
从乾开始顺时针（按原文"交相重"规则）：
乾(1,1)→姤(1,5)→大过(2,5)→鼎(3,5)→恒(4,5)→巽(5,5)→井(6,5)→蛊(7,5)→升(8,5)→
讼(1,6)→困(2,6)→未济(3,6)→解(4,6)→涣(5,6)→坎(6,6)→蒙(7,6)→师(8,6)→
遁(1,7)→咸(2,7)→旅(3,7)→小过(4,7)→渐(5,7)→蹇(6,7)→艮(7,7)→谦(8,7)→
否(1,8)→萃(2,8)→晋(3,8)→豫(4,8)→观(5,8)→比(6,8)→剥(7,8)→坤(8,8)→
**复**(8,4)→颐(7,4)→屯(6,4)→益(5,4)→震(4,4)→噬嗑(3,4)→随(2,4)→无妄(1,4)→
明夷(8,3)→贲(7,3)→既济(6,3)→家人(5,3)→丰(4,3)→离(3,3)→革(2,3)→同人(1,3)→
临(8,2)→损(7,2)→节(6,2)→中孚(5,2)→归妹(4,2)→睽(3,2)→兑(2,2)→履(1,2)→
泰(8,1)→大畜(7,1)→需(6,1)→小畜(5,1)→大壮(4,1)→大有(3,1)→夬(2,1)→回到乾

剔除四正卦（乾、坤、坎、离）后得到60卦序。

**历史对应示例**：

| 公历年份 | 干支 | 岁卦 |
|---------|------|------|
| 1984年 | 甲子 | **复** |
| 2024年 | 甲辰 | 未济 |
| 2025年 | 乙巳 | 解 |
| 2026年 | 丙午 | 涣 |
| 2044年 | 甲子 | **复**（60年循环） |

**计算公式**：
```typescript
// 锚点：甲子年(1984) = 复卦（60卦序索引30）
const JIAZI_YEAR = 1984
const JIAZI_INDEX = 30 // 复卦

// 计算该年的干支索引（0-59）
let ganzhiIndex = (targetYear - JIAZI_YEAR) % 60
if (ganzhiIndex < 0) ganzhiIndex += 60

// 计算在60卦序中的索引（正向循环）
const index = (JIAZI_INDEX + ganzhiIndex) % 60

// 返回先天60卦序中对应索引的卦
return XIANTIAN_60_SEQUENCE[index]
```

### 6.6 月卦（值月卦）

**适用层级**：每月一卦

**锚点规则**："日甲月子，合乎为复"
- 甲子年子月（1984年12月）= 复卦
- 与岁卦使用相同的锚点原则
- 60个月为一个完整循环

**计算规则**：
1. 按先天六十四卦次序循环（剔除四正卦后60卦）
2. 每月对应一个卦，按先天卦序正向循环
3. 锚点：1984年12月（甲子年子月）= 复卦（先天60卦序索引29）

**与岁卦的关系**：
- 岁卦和月卦使用同一套先天60卦循环
- 月卦独立循环，每60个月重复

**计算公式**：
```typescript
// 锚点：1984年12月（甲子年子月）= 复卦
const ANCHOR_YEAR = 1984
const ANCHOR_MONTH = 12
const JIAZI_INDEX = 29 // 复卦

// 计算从锚点到目标日期的月份差
const anchorTotalMonths = ANCHOR_YEAR * 12 + ANCHOR_MONTH
const targetTotalMonths = targetYear * 12 + targetMonth
let monthDiff = targetTotalMonths - anchorTotalMonths

// 计算在60卦序中的索引（正向循环）
let index = (JIAZI_INDEX + monthDiff) % 60
if (index < 0) index += 60

// 返回先天60卦序中对应索引的卦
return XIANTIAN_60_SEQUENCE[index]
```

### 6.7 日卦（值日卦）

**适用层级**：每日一卦

**锚点规则**："日甲月子，合乎为复"
- 甲子日 = 复卦（与甲子年相同）
- 日卦与岁卦使用相同的对应规则
- 60天为一个完整循环

**计算规则**：
1. 按先天六十四卦次序循环（剔除四正卦后60卦）
2. 每日对应一个卦，与六十甲子日一一对应
3. 锚点：甲子日 = 复卦（先天60卦序索引29）

**与六十甲子的对应**：
- 日卦与日干支（六十甲子）一一对应
- 甲子(0) → 复卦, 乙丑(1) → 姤卦, ...
- 60天为一个完整循环

**计算公式**：
```typescript
// 锚点：甲子日 = 复卦（60卦序索引29）
const JIAZI_INDEX = 29

// 基准日：2000年1月1日 = 戊午日（干支索引54）
const BASE_DATE = new Date(2000, 0, 1)
const BASE_INDEX = 54 // 戊午

// 计算与基准日的天数差
const diffDays = Math.round((targetDate - BASE_DATE) / (1000 * 60 * 60 * 24))

// 日干支索引（0-59）
let dayGanZhiIndex = (BASE_INDEX + diffDays) % 60
if (dayGanZhiIndex < 0) dayGanZhiIndex += 60

// 计算在60卦序中的索引（甲子=复卦）
const hexagramIndex = (JIAZI_INDEX + dayGanZhiIndex) % 60

// 返回先天60卦序中对应索引的卦
return XIANTIAN_60_SEQUENCE[hexagramIndex]
```

### 6.8 时卦（值时卦）

**适用层级**：每时辰一卦

**计算规则**：
1. 使用十二消息卦（与月份消息卦相同）
2. 每个时辰对应一个消息卦
3. 与十二地支（时辰）一一对应

**时辰与消息卦对应表**：

| 时辰 | 地支 | 时间范围 | 消息卦 |
|------|-----|---------|--------|
| 子时 | 子 | 23:00 - 01:00 | 復 ䷗ |
| 丑时 | 丑 | 01:00 - 03:00 | 臨 ䷒ |
| 寅时 | 寅 | 03:00 - 05:00 | 泰 ䷊ |
| 卯时 | 卯 | 05:00 - 07:00 | 大壯 ䷡ |
| 辰时 | 辰 | 07:00 - 09:00 | 夬 ䷪ |
| 巳时 | 巳 | 09:00 - 11:00 | 乾 ䷀ |
| 午时 | 午 | 11:00 - 13:00 | 姤 ䷫ |
| 未时 | 未 | 13:00 - 15:00 | 遯 ䷠ |
| 申时 | 申 | 15:00 - 17:00 | 否 ䷋ |
| 酉时 | 酉 | 17:00 - 19:00 | 觀 ䷓ |
| 戌时 | 戌 | 19:00 - 21:00 | 剝 ䷖ |
| 亥时 | 亥 | 21:00 - 23:00 | 坤 ䷁ |

**阴阳消长规律**：
- 子时一阳生（復卦）→ 阳气渐长
- 巳时纯阳（乾卦）→ 午时一阴生（姤卦）
- 亥时纯阴（坤卦）→ 子时阳气复生

**代码实现**：

```typescript
const TWELVE_SOVEREIGN_HEXAGRAMS = [
  1, 3, 7, 15, 31, 63, 62, 60, 56, 48, 32, 0
]

function getShiChenHexagram(hourBranchIndex: number): Hexagram64 {
  const binary = TWELVE_SOVEREIGN_HEXAGRAMS[hourBranchIndex % 12]
  return getHexagram64(binary)
}
```

### 6.9 六套卦象系统对比

**层级关系**：
```
会（辟卦）→ 运（先天60卦，每会5卦×6运）→ 世（运卦爻变）
```

**四正卦处理规则**：

| 层级 | 主序列 | 四正卦用途 |
|------|--------|----------|
| 运、世 | 60正卦（剔除四正卦） | 另作**闰卦**，分主24节气 |
| 年、月、日 | 先天60卦（剔除四正卦） | - |
| 时 | 十二消息卦（含乾坤） | - |

**主序列 vs 闰卦**：
- **主序列**：60正卦，配满360运
- **闰卦**：四正卦（离→乾→坎→坤），分主二十四气的"关节点"

| 系统 | 层级 | 计算方式 | 时长/卦 |
|------|------|---------|---------|
| **运卦** | 会→运 | 先天60卦按序分配，每会5卦×6运 | 2160年 |
| **世卦** | 运→世 | 运卦爻变（剔除四正卦），每2世1卦 | 60年 |
| **闰卦** | 节气 | 离→乾→坎→坤循环 | 分主24气 |
| **岁卦** | 每年 | 先天60卦循环（甲子=复卦） | 1年 |
| **月卦** | 每月 | 先天60卦循环 | 1月 |
| **日卦** | 每日 | 先天60卦循环（甲子日=复卦） | 1日 |
| **时卦** | 每时辰 | 十二消息卦 | 2小时 |

⚠️ **注意**：岁卦/月卦/日卦的"先天60卦循环"可能是后人推演，皇极经世书原文未明确规定。

### 6.10 代码实现

```typescript
// 运卦计算（先天60卦按序分配）
function getYunHexagram(huiIndex: number, yunInHui: number): Hexagram64 {
  // 每会5卦，计算该会在60卦序中的起始位置
  const huiStartIndex = huiIndex * 5
  
  // 每卦统领6运，计算该运对应的卦在会内的位置（0-4）
  const hexagramIndexInHui = Math.floor(yunInHui / 6)
  
  // 计算在60卦序中的全局索引
  const globalIndex = huiStartIndex + hexagramIndexInHui
  
  // 获取对应的卦
  return getHexagram64(XIANTIAN_60_SEQUENCE_FOR_YUN[globalIndex])
}

// 世卦计算（运卦爻变 + 剔除四正卦）
function getShiHexagram(huiIndex: number, yunInHui: number, shiInYun: number): Hexagram64 {
  // 首先获取该运的运卦（父卦）
  const yunHexagram = getYunHexagram(huiIndex, yunInHui)
  
  // 计算需要第几个世卦（每2世共用一个，共6个）
  const shiHexagramIndex = Math.floor((shiInYun % 12) / 2)
  
  // 对运卦进行爻变，生成非四正卦的世卦序列
  const shiSequence: number[] = []
  let yaoIndex = 0
  
  while (shiSequence.length < 6) {
    const yaoPosition = (yaoIndex % 6) + 1
    const changedBinary = changeYao(yunHexagram.binary, yaoPosition)
    
    // 剔除四正卦（乾坤坎离）
    if (!isFourPrincipalHexagram(changedBinary)) {
      shiSequence.push(changedBinary)
    }
    yaoIndex++
  }
  
  return getHexagram64(shiSequence[shiHexagramIndex])
}
```

## 七、皇极经世历的360天制

皇极经世历采用360天/年的制度：
- 12个月 × 30天 = 360天
- 24节气 × 15天 = 360天

而公历是365/366天/年。这个差异通过**节气对齐**来处理：
- 每个节气在皇极历中固定为15天
- 实际上某些节气可能有16-17天
- 皇极日期是理想化的，用于排盘和推演

**重要：** 日干支仍然基于实际的太阳日循环，不受360天制的影响。
