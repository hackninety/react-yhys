# 天干地支计算原理

本文档详细说明皇极经世历与夏历（农历/八字）在天干地支计算上的异同。

## 概述

| 历法 | 换年节点 | 月份起始 | 日干支 | 时干支 |
|------|---------|---------|--------|--------|
| **皇极经世历** | 冬至 | 冬至起子月 | 六十甲子循环 | 日上起时法 |
| **夏历（八字）** | 立春 | 大雪起子月 | 六十甲子循环 | 日上起时法 |

---

## 一、年干支

### 1.1 皇极经世历

**换年节点：冬至**

皇极经世历以冬至为新年的起点。冬至之后即进入下一年。

```
示例：
- 2025年12月21日（冬至）之前 → 乙巳年（皇极第69042年）
- 2025年12月21日（冬至）及之后 → 丙午年（皇极第69043年）
```

**计算方法：**
```typescript
// 判断是否在冬至之后
const dongzhi = getTermStartDate(gregorianYear, 23) // 冬至节气索引=23
const isAfterDongzhi = currentDate >= dongzhi

// 皇极年对应的公历年
const huangjiGregorianYear = isAfterDongzhi ? gregorianYear + 1 : gregorianYear

// 年干支（1984年为甲子年）
const yearIndex = (huangjiGregorianYear - 1984) % 60
const yearGanZhi = 天干[yearIndex % 10] + 地支[yearIndex % 12]
```

### 1.2 夏历（八字）

**换年节点：立春**

夏历（八字命理）以立春为新年的起点。立春之后即进入下一年。

```
示例：
- 2025年2月3日（立春）之前 → 甲辰年
- 2025年2月3日（立春）及之后 → 乙巳年
```

**计算方法：**
```typescript
// 获取该年立春日期（节气索引2=立春）
const lichun = getTermStartDate(year, 2)

// 判断是否在立春之前
const ganzhiYear = currentDate < lichun ? year - 1 : year

// 年干支（1984年为甲子年）
const yearIndex = (ganzhiYear - 1984) % 60
const yearGanZhi = 天干[yearIndex % 10] + 地支[yearIndex % 12]
```

## 二、月干支

### 2.1 皇极经世历

**月份起始：冬至起子月**

皇极经世历以冬至为子月的开始，每两个节气为一个月。

| 皇极月 | 起始节气 | 终止节气 |
|--------|---------|---------|
| 子月（第1月） | 冬至 | 小寒结束 |
| 丑月（第2月） | 大寒 | 立春结束 |
| 寅月（第3月） | 雨水 | 惊蛰结束 |
| 卯月（第4月） | 春分 | 清明结束 |
| 辰月（第5月） | 谷雨 | 立夏结束 |
| 巳月（第6月） | 小满 | 芒种结束 |
| 午月（第7月） | 夏至 | 小暑结束 |
| 未月（第8月） | 大暑 | 立秋结束 |
| 申月（第9月） | 处暑 | 白露结束 |
| 酉月（第10月） | 秋分 | 寒露结束 |
| 戌月（第11月） | 霜降 | 立冬结束 |
| 亥月（第12月） | 小雪 | 大雪结束 |

**月干计算（年上起月法）：**

根据皇极年干确定寅月的天干，然后顺推：

| 年干 | 寅月天干 | 口诀 |
|------|---------|------|
| 甲、己 | 丙 | 甲己之年丙作首 |
| 乙、庚 | 戊 | 乙庚之岁戊为头 |
| 丙、辛 | 庚 | 丙辛必定寻庚起 |
| 丁、壬 | 壬 | 丁壬壬位顺行流 |
| 戊、癸 | 甲 | 戊癸之年甲寅始 |

```
示例（皇极丙午年）：
- 年干=丙 → 寅月起庚
- 子月相对寅月偏移10 → 月干=(庚+10)%10=庚
- 子月月干支=庚子
```

### 2.2 夏历（八字）

**月份起始：以"节"分月（大雪起子月）**

夏历以12个"节"（非"气"）作为月份的分界点：

| 夏历月 | 起始节气（节） | 月支 |
|--------|--------------|------|
| 子月 | 大雪 | 子 |
| 丑月 | 小寒 | 丑 |
| 寅月 | 立春 | 寅 |
| 卯月 | 惊蛰 | 卯 |
| 辰月 | 清明 | 辰 |
| 巳月 | 立夏 | 巳 |
| 午月 | 芒种 | 午 |
| 未月 | 小暑 | 未 |
| 申月 | 立秋 | 申 |
| 酉月 | 白露 | 酉 |
| 戌月 | 寒露 | 戌 |
| 亥月 | 立冬 | 亥 |

**月干计算：** 同皇极，使用年上起月法，但基于夏历年干。

```
示例（夏历乙巳年，大雪后）：
- 年干=乙 → 寅月起戊
- 子月相对寅月偏移10 → 月干=(戊+10)%10=戊
- 子月月干支=戊子
```

### 2.3 皇极与夏历月份对比

以2025年12月23日（冬至后第3天）为例：

| 历法 | 年干支 | 月干支 | 说明 |
|------|-------|--------|------|
| **皇极** | 丙午 | 庚子 | 冬至后=丙午年，冬至起子月，丙年寅月起庚 |
| **夏历** | 乙巳 | 戊子 | 立春前=乙巳年，大雪后=子月，乙年寅月起戊 |

## 三、日干支

### 日干支计算（两种历法一致）

日干支是连续的六十甲子循环，**与历法无关**，是天文周期。

**计算方法：**
```typescript
// 基准日：2000年1月1日 = 戊午日（索引54）
const BASE_DATE = new Date(2000, 0, 1)
const BASE_INDEX = 54

// 计算与基准日的天数差
const diffDays = Math.floor((targetDate - BASE_DATE) / (1000 * 60 * 60 * 24))

// 日干支索引
const dayIndex = (BASE_INDEX + diffDays) % 60
const dayGanZhi = 天干[dayIndex % 10] + 地支[dayIndex % 12]
```

**特点：**
- 日干支是连续循环的，不受年月划分影响
- 皇极经世历和夏历的日干支**完全一致**
- 这是因为日干支记录的是"太阳日"的周期

## 四、时干支

### 时辰划分（两种历法一致）

| 时辰 | 地支 | 时间范围 |
|------|-----|---------|
| 子时 | 子 | 23:00 - 01:00 |
| 丑时 | 丑 | 01:00 - 03:00 |
| 寅时 | 寅 | 03:00 - 05:00 |
| 卯时 | 卯 | 05:00 - 07:00 |
| 辰时 | 辰 | 07:00 - 09:00 |
| 巳时 | 巳 | 09:00 - 11:00 |
| 午时 | 午 | 11:00 - 13:00 |
| 未时 | 未 | 13:00 - 15:00 |
| 申时 | 申 | 15:00 - 17:00 |
| 酉时 | 酉 | 17:00 - 19:00 |
| 戌时 | 戌 | 19:00 - 21:00 |
| 亥时 | 亥 | 21:00 - 23:00 |

### 时干计算（日上起时法）

根据日干确定子时的天干，然后顺推：

| 日干 | 子时天干 | 口诀 |
|------|---------|------|
| 甲、己 | 甲 | 甲己还加甲 |
| 乙、庚 | 丙 | 乙庚丙作初 |
| 丙、辛 | 戊 | 丙辛从戊起 |
| 丁、壬 | 庚 | 丁壬庚子居 |
| 戊、癸 | 壬 | 戊癸何方发，壬子是真途 |

```
示例（丙寅日，卯时 05:00-07:00）：
- 日干=丙 → 子时起戊
- 卯时相对子时偏移3 → 时干=(戊+3)%10=辛
- 卯时时干支=辛卯
```

**注意：** 23:00后的子时属于次日，使用次日的日干计算。

## 五、月卦（十二消息卦）

### 5.1 十二消息卦概述

十二消息卦（又称十二辟卦）是《易经》中表示阴阳消长变化的十二个卦象，对应十二个月份。

"消"指阴气渐长、阳气渐消；"息"指阳气渐长、阴气渐息。

|------|------|------|--------|------|

| 丑月 | 臨 | ䷒ | ⚊⚊⚋⚋⚋⚋ | 二阳 |
| 寅月 | 泰 | ䷊ | ⚊⚊⚊⚋⚋⚋ | 三阳（三阳开泰） |
| 卯月 | 大壯 | ䷡ | ⚊⚊⚊⚊⚋⚋ | 四阳 |
| 辰月 | 夬 | ䷪ | ⚊⚊⚊⚊⚊⚋ | 五阳 |
| 巳月 | 乾 | ䷀ | ⚊⚊⚊⚊⚊⚊ | 六阳（纯阳） |
| 午月 | 姤 | ䷫ | ⚋⚊⚊⚊⚊⚊ | 一阴生（夏至） |

| 亥月 | 坤 | ䷁ | ⚋⚋⚋⚋⚋⚋ | 六阴（纯阴） |

### 5.3 阴阳消长规律

```
冬至（子月）：一阳生于下 → 復卦
     ↓ 阳气渐长
夏至（午月）：纯阳至极 → 乾卦 → 一阴生于下 → 姤卦
     ↓ 阴气渐长
冬至（亥月末）：纯阴至极 → 坤卦 → 一阳复生 → 復卦
```

这体现了《易经》"物极必反、阴阳循环"的哲学思想。

## 六、运卦与世卦

皇极经世书中有两套不同层级的卦象推演系统：**运卦（星卦）**和**世卦（辰卦）**。

### 6.1 时间体系

| 层级 | 名称 | 数量/元 | 时长 |
|------|------|---------|------|
| 日（元） | 元 | 1 | 129600年 |
| 月（会） | 会 | 12 | 10800年/会 |
| 星（运） | 运 | 360 | 360年/运 |
| 辰（世） | 世 | 4320 | 30年/世 |
| 年（岁） | 岁 | 129600 | 1年/岁 |

### 6.2 四正卦的特殊处理

根据《皇极经世书解》原文，四正卦（乾、坤、坎、离）在元会运世体系中有特殊地位：

**核心原则**：
- **主序列剔除**：64卦共384爻，减去四正卦的24爻（4卦×6爻），得到360用数
- **四正卦不是弃用**：而是**另作"闰卦/闰爻"体系**，用来分主二十四气

**原文依据**（《皇极经世书解》卷十三）：
> "乾坤离坎的二十四策拿出去，剩下的就成360"
> "不用者，非不用也，用之以作闰卦"
> "以四闰卦分主二十四气，始以离，继以乾坎，终以坤"

**闰卦对应节气**：

| 闰卦 | 分主节气 | 说明 |
|------|---------|------|
| 离 | 春分、清明、谷雨、立夏、小满、芒种 | 始以离 |
| 乾 | 夏至、小暑、大暑、立秋、处暑、白露 | 继以乾 |
| 坎 | 秋分、寒露、霜降、立冬、小雪、大雪 | 继以坎 |
| 坤 | 冬至、小寒、大寒、立春、雨水、惊蛰 | 终以坤 |

### 6.3 运卦（星卦）

**适用层级**：会 → 运（每会30运）

**计算规则**：
1. 每会以该会的辟卦（消息卦）为基础
2. 通过爻变产生运卦（依次变初爻、二爻...上爻）
3. **主序列剔除四正卦**（四正卦另作闰卦）
4. 循环爻变直至凑满30运（会产生5或6个非四正卦，循环填满）

> **注**：不同会产生的非四正卦数不同（见下表），通过循环重复来凑满30运。

**爻变产生四正卦时的处理**：

| 会 | 辟卦 | 爻变产生四正卦 | 非四正卦数 |
|---|------|---------------|-----------|
| 子 | 复(000001) | 初爻变→坤 | 5 |
| 丑 | 临(000011) | 无 | 6 |
| 寅 | 泰(000111) | 无 | 6 |
| 卯 | 大壮(001111) | 无 | 6 |
| 辰 | 夬(011111) | 上爻变→乾 | 5 |
| 巳 | 乾(111111) | 无 | 6 |
| 午 | 姤(111110) | 初爻变→乾 | 5 |
| 未 | 遁(111100) | 无 | 6 |
| 申 | 否(111000) | 无 | 6 |
| 酉 | 观(110000) | 无 | 6 |
| 戌 | 剥(100000) | 上爻变→坤 | 5 |
| 亥 | 坤(000000) | 无 | 6 |

**午会（姤卦）运卦序列示例**：
```
运序: 181 182 183 184 185 | 186 187 188 189 190 | ... | 206 207 208 209 210
卦名: 遯  讼  巽  鼎  大过 | 遯  讼  巽  鼎  大过 | ... | 遯  讼  巽  鼎  大过
轮次:    第1轮(跳过乾)    |       第2轮        |           第6轮
```

### 6.4 世卦（辰卦）

**适用层级**：运 → 世（每运12世）

**核心原则**："父生子，子生孙"
- 会（辟卦）→ 运（运卦）→ 世（世卦）
- 世卦由**运卦**爻变产生
- **剔除四正卦**（与运卦相同）

**计算规则**：
1. 首先确定该世所属的运，获取**运卦**
2. 对运卦进行爻变产生世卦
3. 遇到四正卦则跳过，继续爻变
4. 6个非四正卦 × 2世/卦 = 12世

**分配规则**：

| 世在运内序号 | 世卦来源 | 时长 |
|-------------|---------|------|
| 第1-2世 | 运卦爻变（非四正卦）第1个 | 60年 |
| 第3-4世 | 运卦爻变（非四正卦）第2个 | 60年 |
| 第5-6世 | 运卦爻变（非四正卦）第3个 | 60年 |
| 第7-8世 | 运卦爻变（非四正卦）第4个 | 60年 |
| 第9-10世 | 运卦爻变（非四正卦）第5个 | 60年 |
| 第11-12世 | 运卦爻变（非四正卦）第6个 | 60年 |

**大过运（午会第5运）世卦示例**：

大过卦（011110）爻变产生世卦（剔除四正卦）：

| 世序号 | 爻变 | 结果 | 是否四正卦 | 世卦 |
|--------|------|------|-----------|------|
| 第1-2世 | 初爻变 | 夬 | 否 | ✓ 夬 |
| 第3-4世 | 二爻变 | 咸 | 否 | ✓ 咸 |
| 第5-6世 | 三爻变 | 困 | 否 | ✓ 困 |
| 第7-8世 | 四爻变 | 井 | 否 | ✓ 井 |
| 第9-10世 | 五爻变 | 恒 | 否 | ✓ 恒 |
| 第11-12世 | 上爻变 | 姤 | 否 | ✓ 姤 |

### 6.5 岁卦（值年卦）

**适用层级**：每年一卦

**计算规则**：
1. 按先天六十四卦次序循环（剔除四正卦后60卦）
2. 每年对应一个卦，按先天卦序逆向循环
3. 锚点：公历2025年 = 革卦（先天60卦序第17位）

**先天卦序（邵雍方圆图）**：
从乾开始顺时针：乾→夬→大有→大壮→小畜→需→大畜→泰→履→兑→睽→归妹→中孚→节→损→临→**同人**→**革**→离→**丰**→家人→既济→贲→明夷→无妄→随→噬嗑→震→益→屯→颐→复→姤→大过→鼎→恒→巽→井→蛊→升→讼→困→未济→解→涣→坎→蒙→师→遁→咸→旅→小过→渐→蹇→艮→谦→否→萃→晋→豫→观→比→剥→坤

剔除四正卦（乾、坤、坎、离）后得到60卦序。

**历史对应示例**：

| 公历年份 | 先天60卦序位置 | 岁卦 |
|---------|--------------|------|
| 2024年 | 第18位 | **丰** |
| 2025年 | 第17位 | **革** |
| 2026年 | 第16位 | **同人** |
| 2027年 | 第15位 | 临 |

**计算公式**：
```typescript
// 锚点：2025年 = 革卦（60卦序第17位，索引16）
const ANCHOR_YEAR = 2025
const ANCHOR_INDEX = 16

// 计算年份差（逆向循环）
const yearDiff = ANCHOR_YEAR - targetYear
let index = (ANCHOR_INDEX + yearDiff) % 60
if (index < 0) index += 60

// 返回先天60卦序中对应索引的卦
return XIANTIAN_60_SEQUENCE[index]
```

### 6.6 月卦（值月卦）

**适用层级**：每月一卦

**计算规则**：
1. 按先天六十四卦次序循环（剔除四正卦后60卦）
2. 每月对应一个卦，按先天卦序逆向循环
3. 锚点：公历2025年12月 = 同人卦（先天60卦序第16位）

**与岁卦的关系**：
- 岁卦和月卦使用同一套先天60卦循环
- 月卦继续在岁卦的基础上按月递增

**历史对应示例**：

| 公历年月 | 先天60卦序位置 | 月卦 |
|---------|--------------|------|
| 2025年12月 | 第16位 | **同人** |
| 2026年1月 | 第15位 | **临** |
| 2026年2月 | 第14位 | 损 |
| 2026年3月 | 第13位 | 节 |

**计算公式**：
```typescript
// 锚点：2025年12月 = 同人卦（60卦序第16位，索引15）
const ANCHOR_YEAR = 2025
const ANCHOR_MONTH = 12
const ANCHOR_INDEX = 15

// 计算总月份差
const anchorTotalMonths = ANCHOR_YEAR * 12 + ANCHOR_MONTH
const targetTotalMonths = targetYear * 12 + targetMonth
const monthDiff = anchorTotalMonths - targetTotalMonths

// 计算在60卦序中的索引（逆向循环）
let index = (ANCHOR_INDEX + monthDiff) % 60
if (index < 0) index += 60

// 返回先天60卦序中对应索引的卦
return XIANTIAN_60_SEQUENCE[index]
```

### 6.7 日卦（值日卦）

**适用层级**：每日一卦

**计算规则**：
1. 按先天六十四卦次序循环（剔除四正卦后60卦）
2. 每日对应一个卦，与六十甲子日一一对应
3. 锚点：甲子日 = 夬卦（先天60卦序第1位）

**与六十甲子的对应**：
- 日卦与日干支（六十甲子）一一对应
- 甲子(0) → 夬卦, 乙丑(1) → 大有卦, ...
- 60天为一个完整循环

**计算公式**：
```typescript
// 基准日：2000年1月1日 = 戊午日（索引54）
const BASE_DATE = new Date(2000, 0, 1)
const BASE_INDEX = 54 // 戊午

// 计算与基准日的天数差
const diffDays = Math.round((targetDate - BASE_DATE) / (1000 * 60 * 60 * 24))

// 日干支索引（0-59）
let dayGanZhiIndex = (BASE_INDEX + diffDays) % 60
if (dayGanZhiIndex < 0) dayGanZhiIndex += 60

// 返回先天60卦序中对应索引的卦
return XIANTIAN_60_SEQUENCE[dayGanZhiIndex]
```

### 6.8 时卦（值时卦）

**适用层级**：每时辰一卦

**计算规则**：
1. 使用十二消息卦（与月份消息卦相同）
2. 每个时辰对应一个消息卦
3. 与十二地支（时辰）一一对应

**时辰与消息卦对应表**：

| 时辰 | 地支 | 时间范围 | 消息卦 |
|------|-----|---------|--------|
| 子时 | 子 | 23:00 - 01:00 | 復 ䷗ |
| 丑时 | 丑 | 01:00 - 03:00 | 臨 ䷒ |
| 寅时 | 寅 | 03:00 - 05:00 | 泰 ䷊ |
| 卯时 | 卯 | 05:00 - 07:00 | 大壯 ䷡ |
| 辰时 | 辰 | 07:00 - 09:00 | 夬 ䷪ |
| 巳时 | 巳 | 09:00 - 11:00 | 乾 ䷀ |
| 午时 | 午 | 11:00 - 13:00 | 姤 ䷫ |
| 未时 | 未 | 13:00 - 15:00 | 遯 ䷠ |
| 申时 | 申 | 15:00 - 17:00 | 否 ䷋ |
| 酉时 | 酉 | 17:00 - 19:00 | 觀 ䷓ |
| 戌时 | 戌 | 19:00 - 21:00 | 剝 ䷖ |
| 亥时 | 亥 | 21:00 - 23:00 | 坤 ䷁ |

**阴阳消长规律**：
- 子时一阳生（復卦）→ 阳气渐长
- 巳时纯阳（乾卦）→ 午时一阴生（姤卦）
- 亥时纯阴（坤卦）→ 子时阳气复生

**代码实现**：

### 6.9 六套卦象系统对比

**"父生子，子生孙"层级关系**：
```
会（辟卦）
    ↓ 爻变（主序列剔除四正卦）
运（运卦）
    ↓ 爻变（主序列剔除四正卦）
世（世卦）
```

**四正卦处理规则**：

| 层级 | 主序列 | 四正卦用途 |
|------|--------|----------|
| 元、会、运、世 | 60正卦（剔除四正卦） | 另作**闰卦**，分主24节气 |
| 年、月、日 | 先天60卦（剔除四正卦） | - |
| 时 | 十二消息卦（含乾坤） | - |

**主序列 vs 闰卦**：
- **主序列**：60正卦，配满360运/360世等主时间轴
- **闰卦**：四正卦（离→乾→坎→坤），分主二十四气的"关节点"

| 系统 | 层级 | 基础卦 | 计算方式 | 时长/卦 |
|------|------|--------|---------|---------|
| **运卦** | 会→运 | 辟卦 | 爻变（主序列60卦） | 2160年 |
| **世卦** | 运→世 | **运卦** | 爻变（主序列60卦） | 60年 |
| **闰卦** | 节气 | 四正卦 | 离→乾→坎→坤 | 分主24气 |
| **岁卦** | 每年 | - | 先天60卦循环 | 1年 |
| **月卦** | 每月 | - | 先天60卦循环 | 1月 |
| **日卦** | 每日 | - | 先天60卦循环 | 1日 |
| **时卦** | 每时辰 | - | 十二消息卦 | 2小时 |

⚠️ **注意**：岁卦/月卦/日卦的"先天60卦循环"可能是后人推演，皇极经世书原文未明确规定。时卦使用十二消息卦是合理推演。

### 6.10 代码实现

```typescript
// 运卦计算（剔除四正卦）
function getYunHexagram(huiIndex: number, yunInHui: number): Hexagram64 {
  const sovereignBinary = TWELVE_SOVEREIGN_HEXAGRAMS[huiIndex % 12]
  const yunSequence: number[] = []
  let yaoIndex = 0
  
  while (yunSequence.length < 30) {
    const yaoPosition = (yaoIndex % 6) + 1
    const changedBinary = changeYao(sovereignBinary, yaoPosition)
    
    // 剔除四正卦（乾坤坎离）
    if (!isFourPrincipalHexagram(changedBinary)) {
      yunSequence.push(changedBinary)
    }
    yaoIndex++
  }
  
  return getHexagram64(yunSequence[yunInHui % 30])
}

// 世卦计算（"父生子，子生孙"原则 + 剔除四正卦）
function getShiHexagram(huiIndex: number, yunInHui: number, shiInYun: number): Hexagram64 {
  // 首先获取该运的运卦（父卦）
  const yunHexagram = getYunHexagram(huiIndex, yunInHui)
  
  // 计算需要第几个世卦（每2世共用一个，共6个）
  const shiHexagramIndex = Math.floor((shiInYun % 12) / 2)
  
  // 对运卦进行爻变，生成非四正卦的世卦序列
  const shiSequence: number[] = []
  let yaoIndex = 0
  
  while (shiSequence.length < 6) {
    const yaoPosition = (yaoIndex % 6) + 1
    const changedBinary = changeYao(yunHexagram.binary, yaoPosition)
    
    // 剔除四正卦（乾坤坎离）
    if (!isFourPrincipalHexagram(changedBinary)) {
      shiSequence.push(changedBinary)
    }
    yaoIndex++
  }
  
  return getHexagram64(shiSequence[shiHexagramIndex])
}
```

## 七、皇极经世历的360天制

皇极经世历采用360天/年的制度：
- 12个月 × 30天 = 360天
- 24节气 × 15天 = 360天

而公历是365/366天/年。这个差异通过**节气对齐**来处理：
- 每个节气在皇极历中固定为15天
- 实际上某些节气可能有16-17天
- 皇极日期是理想化的，用于排盘和推演

**重要：** 日干支仍然基于实际的太阳日循环，不受360天制的影响。
