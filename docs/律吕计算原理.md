# 律吕计算原理

本文档详细说明皇极经世书中年月日时与律吕的对应关系。

## 概述

《皇极经世书》将时间划分为两个层次：
- **元会运世**：对应**卦象**（象数）
- **年月日时**：对应**律吕**（声音）

> 原文依据：邵雍认为"声音律吕"与"象数卦爻"互为表里，天地万物的运行可以用这两套系统来描述。

---

## 一、十二律吕

### 1.1 十二律吕概述

十二律吕是中国古代音乐理论中的十二个音律，分为**六律**（阳）和**六吕**（阴）：

| 类型 | 律吕名称 | 音高顺序 |
|------|---------|---------|
| **六律（阳）** | 黄钟、太簇、姑洗、蕤宾、夷则、无射 | 1、3、5、7、9、11 |
| **六吕（阴）** | 大吕、夹钟、仲吕、林钟、南吕、应钟 | 2、4、6、8、10、12 |

> **注**：年律/日律仅使用前**五正律**（黄钟、太簇、姑洗、蕤宾、夷则），"无射"不参与天干对应。

### 1.2 十二律吕完整序列

按音高从低到高排列：

| 序号 | 律吕名 | 类型 | 对应地支 | 月份 |
|------|--------|------|---------|------|
| 1 | 黄钟 | 律 | 子 | 十一月 |
| 2 | 大吕 | 吕 | 丑 | 十二月 |
| 3 | 太簇 | 律 | 寅 | 正月 |
| 4 | 夹钟 | 吕 | 卯 | 二月 |
| 5 | 姑洗 | 律 | 辰 | 三月 |
| 6 | 仲吕 | 吕 | 巳 | 四月 |
| 7 | 蕤宾 | 律 | 午 | 五月 |
| 8 | 林钟 | 吕 | 未 | 六月 |
| 9 | 夷则 | 律 | 申 | 七月 |
| 10 | 南吕 | 吕 | 酉 | 八月 |
| 11 | 无射 | 律 | 戌 | 九月 |
| 12 | 应钟 | 吕 | 亥 | 十月 |

## 二、年与律吕

### 2.1 年律计算规则

年的律吕按**天干**循环（10干分5对），使用**五正律**（阳律中的前五律）对应：

| 天干 | 对应律 | 说明 |
|------|--------|------|
| 甲、己 | 黄钟 | 甲己合化土，黄钟为律首 |
| 乙、庚 | 太簇 | 乙庚合化金 |
| 丙、辛 | 姑洗 | 丙辛合化水 |
| 丁、壬 | 蕤宾 | 丁壬合化木 |
| 戊、癸 | 夷则 | 戊癸合化火 |

### 2.2 年律计算示例

```
2025年（乙巳年）：
- 年干 = 乙
- 乙对应太簇
- 年律 = 太簇

2026年（丙午年）：
- 年干 = 丙
- 丙对应姑洗
- 年律 = 姑洗
```

## 三、月与律吕

### 3.1 月律计算规则

月的律吕按十二地支对应十二律吕：

| 月支 | 对应律吕 | 皇极月序 |
|------|---------|---------|
| 子 | 黄钟 | 第1月 |
| 丑 | 大吕 | 第2月 |
| 寅 | 太簇 | 第3月 |
| 卯 | 夹钟 | 第4月 |
| 辰 | 姑洗 | 第5月 |
| 巳 | 仲吕 | 第6月 |
| 午 | 蕤宾 | 第7月 |
| 未 | 林钟 | 第8月 |
| 申 | 夷则 | 第9月 |
| 酉 | 南吕 | 第10月 |
| 戌 | 无射 | 第11月 |
| 亥 | 应钟 | 第12月 |

### 3.2 月律计算示例

```
皇极子月（冬至后）：
- 月支 = 子
- 月律 = 黄钟

皇极寅月（雨水后）：
- 月支 = 寅
- 月律 = 太簇
```

## 四、日与律吕

### 4.1 日律计算规则

日的律吕按六十甲子循环，有两种对应方式：

**方式一：按天干对应六律**（与年律相同）

| 日干 | 对应律 |
|------|--------|
| 甲、己 | 黄钟 |
| 乙、庚 | 太簇 |
| 丙、辛 | 姑洗 |
| 丁、壬 | 蕤宾 |
| 戊、癸 | 夷则 |

**方式二：按地支对应十二律吕**

| 日支 | 对应律吕 |
|------|---------|
| 子 | 黄钟 |
| 丑 | 大吕 |
| 寅 | 太簇 |
| ... | ... |
| 亥 | 应钟 |

### 4.2 本项目采用方式

本项目采用**方式一**（按天干对应六律），与年律保持一致。

## 六、律吕与卦象的关系

### 6.1 对应层级

| 层级 | 象数系统 | 声音系统 |
|------|---------|---------|
| 元 | 卦象 | - |
| 会 | 卦象（辟卦） | - |
| 运 | 卦象（运卦） | - |
| 世 | 卦象（世卦） | - |
| 年 | 卦象（岁卦） | 律吕（年律） |
| 月 | 卦象（月卦） | 律吕（月律） |
| 日 | 卦象（日卦） | 律吕（日律） |
| 时 | 卦象（时卦） | 律吕（时律） |

### 6.2 完整四柱律吕

以2025年12月24日10时为例：

| 柱 | 干支 | 卦象 | 律吕 |
|----|------|------|------|
| 年柱 | 丙午年 | ䷌同人（岁卦） | 姑洗 |
| 月柱 | 庚子月 | ䷒臨（值月卦） | 黄钟 |
| 日柱 | 戊辰日 | ䷰革（值日卦） | 夷则 |
| 时柱 | 丁巳时 | ䷀乾（消息卦） | 仲吕 |

> **注**：月卦此处指"值月卦"（先天60卦循环），非"十二消息卦"。消息卦中子月应为"復"。

## 七、代码实现

### 7.1 律吕常量

```typescript
// 十二律吕（按地支顺序：子丑寅卯辰巳午未申酉戌亥）
const TWELVE_LVLV = [
  { name: '黄钟', type: '律', pinyin: 'Huáng Zhōng' },
  { name: '大吕', type: '吕', pinyin: 'Dà Lǚ' },
  { name: '太簇', type: '律', pinyin: 'Tài Cù' },
  { name: '夹钟', type: '吕', pinyin: 'Jiā Zhōng' },
  { name: '姑洗', type: '律', pinyin: 'Gū Xiǎn' },
  { name: '仲吕', type: '吕', pinyin: 'Zhòng Lǚ' },
  { name: '蕤宾', type: '律', pinyin: 'Ruí Bīn' },
  { name: '林钟', type: '吕', pinyin: 'Lín Zhōng' },
  { name: '夷则', type: '律', pinyin: 'Yí Zé' },
  { name: '南吕', type: '吕', pinyin: 'Nán Lǚ' },
  { name: '无射', type: '律', pinyin: 'Wú Yì' },
  { name: '应钟', type: '吕', pinyin: 'Yìng Zhōng' },
]

// 天干对应六律（甲己=黄钟，乙庚=太簇，丙辛=姑洗，丁壬=蕤宾，戊癸=夷则）
const STEM_TO_LV = [0, 2, 4, 6, 8, 0, 2, 4, 6, 8]
// 索引：        甲  乙  丙  丁  戊  己  庚  辛  壬  癸
// 注意：六律中的第6个阳律"无射"(10)在此映射中未被直接使用，这是基于五合化气与五正声的对应。
```

### 7.2 律吕计算函数

```typescript
// 根据天干获取律吕（用于年律、日律）
function getLvByTianGan(stemIndex: number): LvLv {
  const lvIndex = STEM_TO_LV[stemIndex % 10]
  return TWELVE_LVLV[lvIndex]
}

// 根据地支获取律吕（用于月律、时律）
function getLvByDiZhi(branchIndex: number): LvLv {
  return TWELVE_LVLV[branchIndex % 12]
}

// 获取年律
function getYearLv(yearStemIndex: number): LvLv {
  return getLvByTianGan(yearStemIndex)
}

// 获取月律
function getMonthLv(monthBranchIndex: number): LvLv {
  return getLvByDiZhi(monthBranchIndex)
}

// 获取日律
function getDayLv(dayStemIndex: number): LvLv {
  return getLvByTianGan(dayStemIndex)
}

// 获取时律
function getHourLv(hourBranchIndex: number): LvLv {
  return getLvByDiZhi(hourBranchIndex)
}
```
