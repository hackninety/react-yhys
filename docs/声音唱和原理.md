# 声音唱和原理

本文档详细说明皇极经世书中"声音唱和图"的基础概念、与天干地支的对应法则，以及本系统的概览展示原理。

## 概述

《声音唱和图》出自邵雍《皇极经世书》卷七至卷十，是其创立的一套复杂而宏大的音韵学系统。黄畿注曰：“声音律吕，圆唱方和，而后乾坤坎离用焉，天地万物之理贯于一矣。”

- **律吕**（黄钟、大吕等）：代表**音高**（Pitch）。
- **声音唱和**（天声、地音）：代表**音韵**（Phonetics），即声母与韵母的拼合发音。

## 一、天声与地音的核心概念

邵雍将汉语音韵拆分为“声”与“音”两大部分，这也是现代音韵学“韵母”与“声母”概念的早期独特体系。

### 1. 天声（韵母部类）
天声共有 **112 个**实际用声（从160个先天数中减去48个不用之数得出）。在这套体系中，天声被分为**十组**，以天干之数字"十"为纲纪。
- **分类维度**：
  - 四声：平、上、去、入（对应日、月、星、辰四象）
  - 洪细：辟（开口大）、翕（合口小）
  - 清浊：清声（属阳）、浊声（属阴）

### 2. 十二地音（声母部类）
地音共有 **152 个**实际用音（从192个本地数中减去40个不用之数得出）。地音被分为**十二组**，以地支之数字"十二"为纲纪。
- **分类维度**：
  - 发音部位（四象对应）：
    - 水（羽音：唇音）
    - 火（徵音：齿音/舌齿音）
    - 土（宫音：喉音/牙音）
    - 石（商音/角音：舌头/半舌半齿音）
  - 呼法：开、发、收、闭
  - 清浊：清（属阳）、浊（属阴）

## 二、字例解读（唱和矩阵）

天声与地音交叉，形成一个庞大的发音字图（部分位置有音无字，以 `○` 标记）。

- **图例解码**：
  - `多` 字：第一声（平声、辟、日象、清） + 第一音（水象、开、清）。
  - `回` 字：第四声（平声、辟、辰象、浊） + 第一音（水象、开、清）。

## 三、干支与声组的对应法则（本系统实现方案）

**注意：严格来说，《声音唱和图》是一张静态音韵分类表，并非一套“动态时间函数”。** 古人不会说“今天是甲子日，所以今天发'多'音”。

但是，为了在日历系统中直观展示这套伟大体系的宏观架构，并遵循邵子“**天数十，地数十二**”的理法，我们以**类比索引**的方式展示当日干支在唱和图中的位置：

### 1. 天干对应十天声组
根据天干的序号（0-9），直接映射到第一声至第十声。
- `甲` (0) → 第一声组（日象平声为主）
- `乙` (1) → 第二声组
- ...
- `癸` (9) → 第十声组

### 2. 地支对应十二地音组
根据地支的序号（0-11），直接映射到第一音至第十二音。
- `子` (0) → 第一音组（水象开清）
- `丑` (1) → 第二音组
- ...
- `亥` (11) → 第十二音组

## 四、显示示例

假设今天是 **甲子日**：
- 天干为**甲**，则属于天声**第一组**。
  - 界面显示：天声第一组（平层·辟·清·日象）
  - 代表字：多、良、千、刀、妻
- 地支为**子**，则属于地音**第一组**。
  - 界面显示：地音第一组（水象·开·清）
  - 代表字：多、内、乃
- 结合查看，此日暗示着唱和图中“第一声×第一音”相交之位（如“多”字）。

## 五、完整 10x12 唱和千字发音矩阵（Phase 2 Step 2a 已实现）

系统已实现独立的**全屏交互矩阵组件** (`ChangheMatrix.tsx`)，通过日期详情弹窗中的"查看完整声音唱和千字发音图"按钮进入。

### 矩阵结构
- **纵轴**：10组天声 x 4行（平上去入） = 40 行
- **横轴**：12组地音（以地支子至亥排列）= 12 列
- **单元格**：每个交叉点对应一个汉字例，空位以 `○` 标记

### 交互特性
- **十字高亮 (Crosshair)**：鼠标悬停时，当前行与列双向高亮
- **岁日自动定位**：打开时自动高亮当日天干对应的天声行（蓝色）和地支对应的地音列（橙色），交汇点以金色强调显示
- **粘性表头**：地音列标题常驻顶部，滚动时不丢失列参照

### 数据文件
矩阵数据源位于 `src/data/changheMatrix.ts`，采用 3 级嵌套结构：
- `TianShengGroup` -> `TianShengRow` -> `ChangheCell[]`

目前已录入所有可从权威文本确认的字例，后续可持续补全完善（Step 2b）。
